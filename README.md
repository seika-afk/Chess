#  Chess Game

A simple 2-player chess game using **Node.js**, **Express**, **Socket.io**, and **chess.js**.

---

##  Setup

```bash
git clone <repo-url>
cd chess
npm install
node app.js
```
## How To Play 

1. Run `npm start`
2. Open a tab and got to `localhost:3000`.
3. It will run as White player.
4. Open Another tab to play as Black Player.
